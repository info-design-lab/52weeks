Movies Revenue Analysis (1900-2017)

Arihant Parsoya

There are ~45000 movies released between 1900 and 2017. This visualization explores how the revenue of movies relate with their runtime, budget, rating and popularity

About the data

This visualization is created using data obtained from Kaggle. The data corresponds to movie profiles in The Movie Database (TMDb).
Due to high humber of movies released since 1900, we are only chosing movies whose

  • Runtime is less than 200 minutes
  • Rating is more than 6.5
Since a movie can have multiple genre, we have only considered one distinguishing genre for a movie. Those are: Action, Drama, Comedy, Horror, Animation and Documentary.
